Acerca de Mí

Mi nombre es Erika González, soy Desarrolladora Web y Diseñadora Gráfica especializada en la Programación Web contando con más de tres años de experiencia laboral. Asimismo, pertenezco a una Agencia de Diseños Web, en la cual ofrecemos servicios de Diseño Gráfico, Desarrollo de Sitios Web y mucho más. No obstante, mi objetivo principal es poder brindarles la mejor experiencia y servicio web a cada cliente; creando Fantásticos Sitios Web con Diseños Hermosos, Funcionales, Adaptables y Eficientes para impulsar su negocio y profesión a un nivel excepcional.

De igual manera, soy Ceo y Fundadora de SoftxNews, un proyecto lanzado en el año 2020, con el fin de crear una organización para brindar servicios de creación de páginas y aplicaciones webs más fáciles, eficientes y accesibles para todos.
